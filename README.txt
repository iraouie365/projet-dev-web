Projet: Gestion des Demandes d'Expression de Besoins
Tech: PHP 7+, MySQL, Bootstrap 5

1. Importer le fichier sql/gestion_besoins.sql dans votre MySQL.
2. Configurer config/db.php (user/mot de passe MySQL).
3. Copier le dossier gestion_besoins dans votre serveur web (htdocs, www, etc.).
4. Accéder à index.php via votre navigateur.
5. Connexion admin: admin@example.com / admin123
